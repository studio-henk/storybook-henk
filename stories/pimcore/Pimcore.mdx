# Pimcore stuff

## CSS in pimcore repository

All CSS partials are in the repository. 

```
public/static/std/assets/css/henk-styles.css
```

When running the npm script "CompileCSS:prod" this file will be compiled and minified to:

```
public/static/std/assets/css/henk-styles.min.css
```

See ```package.json``` for the script and dependencies.

```json
{
  "name": "pimcore-local",
  "version": "1.0.0",
  "description": "henk styles",
  "main": "index.js",
  "scripts": {
    "test": "echo \"Error: no test specified\" && exit 1",
    "compileCSS": "npx postcss public/static/std/assets/css/henk-styles.css --map -o public/static/std/assets/css/henk-styles.min.css"
  },
  "author": "",
  "license": "ISC",
  "devDependencies": {
    "node": "18.18.1",
    "autoprefixer": "^10.4.19",
    "cssnano": "^7.0.1",
    "postcss": "^8.4.38",
    "postcss-cli": "^11.0.0",
    "postcss-discard-comments": "^7.0.0",
    "postcss-import": "^16.1.0"
  }
}
```
